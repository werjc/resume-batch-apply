# 隐私政策

**最后更新**: 2026年7月12日

## 数据收集与使用

"一键批量投递简历"扩展**不收集、不上传、不分享**任何用户数据到第三方服务器。

### 本地存储的数据

以下数据仅存储在您的 Chrome 浏览器本地（`chrome.storage.local`），完全由您控制：

| 数据类型 | 用途 | 是否离开本机 |
|----------|------|-------------|
| API Key（DeepSeek/OpenAI） | 调用 AI 分析接口 | 仅在您点击"AI分析"时发送到您配置的 API 地址 |
| 简历文本 | AI 匹配评分 | 仅在您点击"AI分析"时随请求发送到您配置的 API |
| 投递历史记录 | 统计面板 + 去重 | 否 |
| 筛选偏好 | 记住您的筛选条件 | 否 |
| 定时任务信息 | 定时投递 | 否 |

### 网络请求

- AI 分析：仅在您主动点击"AI分析"时，向您配置的 API 提供商（DeepSeek/OpenAI/自定义）发送请求
- 深度搜索：不进行任何自动网络请求

### 权限说明

扩展请求的权限仅用于核心功能：
- `storage`：本地存储设置和历史
- `alarms`：定时投递任务
- `notifications`：投递完成通知
- `activeTab`：仅在您点击扩展图标时访问当前标签页
- `scripting`：向支持的招聘网站注入投递脚本
- `host_permissions`：仅在 8 个招聘网站和 2 个 API 域名生效

### 数据删除

您可以随时：
1. 在 Chrome 扩展管理页移除扩展（自动清除所有本地数据）
2. 在扩展 AI 面板中清除调试日志
